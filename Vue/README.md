# Getting Started with DevExtreme Vue PivotGrid

This project was bootstrapped with [Vue CLI](https://cli.vuejs.org/).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
